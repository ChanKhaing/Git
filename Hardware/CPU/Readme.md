# 🧠 CPU ဆိုတာ ဘာလဲ?

CPU ကို **ဦးနှောက်** လို့ မှတ်ထားပါ။

CPU က စဉ်းစားတယ်။

CPU က တွက်ချက်တယ်။

# Motherboard က ဘာလဲ?

Motherboard ကို

**လမ်းမကြီး** လို့ စဉ်းစားပါ။

CPU၊ RAM၊ SSD၊ GPU အားလုံးဟာ Motherboard ပေါ်မှာ ချိတ်ဆက်ထားတယ်။

```
        CPU
         │
         │
RAM ─────┼───── SSD
         │
         │
        GPU
         │
     Monitor
```

Motherboard မရှိရင် ဒီအစိတ်အပိုင်းတွေ တစ်ခုနဲ့တစ်ခု မဆက်သွယ်နိုင်ဘူး။

# 🎯 အရေးကြီးဆုံး မှတ်ထားရမယ့် အချက်

Computer တစ်လုံး အလုပ်လုပ်ပုံကို စာတစ်ကြောင်းတည်းနဲ့ ပြောရရင်—

```
SSD က Data ကို သိမ်းထားတယ်။

↓

RAM က အလုပ်လုပ်ဖို့ ယာယီကိုင်ထားတယ်။

↓

CPU က RAM ထဲက Data ကို တွက်ချက်တယ်။

↓

GPU (လိုအပ်ရင်) ပုံဆွဲတယ်။

↓

Monitor က ရလဒ်ကို ပြတယ်။
```

## 🌱 နောက်သင်ရမယ့် အစီအစဉ်

 Linux, C, DevOps, Networking ကို အခြေခံကနေ နက်နက်ရှိုင်းရှိုင်း နားလည်ချင်တယ်ဆိုတော့ နောက်တစ်ဆင့်မှာ ဒီအကြောင်းတွေကို လေ့လာသင့်တယ်။

1. **Motherboard** (CPU, RAM, SSD ဘယ်လိုချိတ်ထားလဲ)
2. **CPU က instruction တစ်ခုကို ဘယ်လို execute လုပ်လဲ**
3. **RAM ရဲ့ အတွင်းပိုင်း (Address, Memory Cell)**
4. **Cache (L1, L2, L3)**
5. **Power Button နှိပ်တဲ့အချိန်ကနေ Windows ပေါ်လာတဲ့အထိ Boot Process**

ဒီအခြေခံ ၅ ခုကို နားလည်သွားရင် Operating System, Linux, C Programming, Networking, DevOps ကို သင်တဲ့အခါ "ဘာကြောင့် ဒီလိုဖြစ်တာလဲ" ဆိုတာကို အလွယ်တကူ ချိတ်ဆက်နားလည်နိုင်လာပါလိမ့်မယ်။
