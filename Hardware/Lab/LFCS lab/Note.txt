# Setup ကို အတတ်နိုင်ဆုံး လွယ်အောင် လုပ်ထားပါတယ်

"git clone" လုပ်ပြီး install script ကို run လိုက်ရုံပါပဲ။

Installer က VM build မလုပ်ခင် CPU virtualization, RAM, Storage, Hyper-V / WSL2 / KVM conflicts, VirtualBox နဲ့ Vagrant ရှိမရှိ အရင်စစ်ပေးပါတယ်။

လိုအပ်တဲ့ tools တွေ မရှိရင် install လုပ်ပေးပြီး၊ VM build ပြီးတဲ့နောက် sample question တစ်ခုကို end-to-end run ပြီး self-verify လုပ်ပေးပါတယ်။ PASS ဖြစ်မှ environment က ready ဖြစ်တယ်လို့ သတ်မှတ်ထားပါတယ်။

# 250 Original Hands-on Questions

Official LFCS domains ၅ ခုလုံးကို cover လုပ်ထားပါတယ်။

- Essential Commands
- Operations & Deployment
- Networking
- Storage
- Users & Groups

စုစုပေါင်း 250 original practice questions (215 single-node + 35 real two-node labs) ပါပြီး NFS, NBD, Reverse Proxy, LDAP, SSH နဲ့ NTP လို services တွေကို real private network ပေါ်မှာ practice လုပ်နိုင်ပါတယ်။

# Practice Mode / Exam Mode

Practice Mode မှာ hint ကြည့်ပြီး ဖြည်းဖြည်းလေ့ကျင့်နိုင်သလို၊ Exam Mode မှာတော့ timer၊ random questions နဲ့ mock exam ပုံစံ practice လုပ်နိုင်ပါတယ်။

# Automatic Validation

Practice လုပ်ပြီး "ငါမှန်လား?" ဆိုတာ မသိရင် အဓိပ္ပာယ်မရှိဘူးထင်လို့ question တိုင်းမှာ validator ထည့်ထားပါတယ်။

File contents, ownership, permissions, services, ports, mounts နဲ့ persistence အထိ စစ်ပြီး RESULT: PASS / FAIL ကို ချက်ချင်းပြပေးပါတယ်။

# Automatic Snapshot Restore

ဒါက ကျွန်တော် အကြိုက်ဆုံးအပိုင်းပါဗျ။

Question တစ်ခုပြီးတိုင်း clean snapshot ကို auto restore လုပ်ပေးတာကြောင့် VM reset ကို လက်နဲ့ပြန်လုပ်စရာ မလိုတော့ပါဘူး။

# Muscle Memory ကို ဦးစားပေးထားပါတယ်

Question အများစုက တူညီတဲ့ skill ကို variation အမျိုးမျိုးနဲ့ ပြန်လေ့ကျင့်နိုင်အောင် ရေးထားပါတယ်။

ရည်ရွယ်ချက်က command တွေ အလွတ်ကျက်ဖို့ မဟုတ်ဘဲ အချိန်ဖိအားအောက်မှာ မြန်မြန်လုပ်နိုင်တဲ့ muscle memory တည်ဆောက်နိုင်ဖို့ပါ။

# ပထမဆုံး setup ကတော့ စိတ်နည်းနည်းရှည်ဖို့လိုပါတယ်။

VM boxes download နဲ့ build လုပ်ရတာကြောင့် ~၁ နာရီဝန်းကျင် ကြာနိုင်ပါတယ်။

Requirements ကတော့

• x86-64 CPU (VT-x / AMD-V)

• RAM 8 GB minimum (16 GB+ recommended)

• Disk ~30 GB minimum (~50 GB recommended)
